# ASCTool

APk签名验证破解工具 APK Signature Crack Tool

本工具只对那些仅通过 PackageManager.getPackageInfo().signatures 来校验签名的应用有效。

基于JavaFx框架，使用Kotlin语言实现
## 更新日志

### 2022年2月18日
1.新增v2和v3签名，与自动对齐（选项可选）
2.兼容mac（虽然未测试）
3.修复找不到默认签名文件问题

### 2021年3月17日
1.新增重签名功能(可对未签名的apk进行签名) 
2.新增文件拖动功能
3.修复使用自定义签名报错问题
